## Overview

A responsive personal portfolio website built with React, Vite, and Tailwind CSS.

The website includes sections for:

- Hero introduction
- About
- Skills
- Services
- Workflow
- Technologies & Tools
- Contact
- Social links

- React
- Vite
- Tailwind CSS
- Font Awesome
- Netlify
